# ODORU STUDIO site

ODORU STUDIO の静的サイト一式です。

## 使い方

GitHub のリポジトリ直下に、以下をそのまま置いてください。

- `index.html`
- `style.css`
- `script.js`
- `app-ads.txt`
- `assets/`
- `.nojekyll`

GitHub Pages を有効化すると、トップページとして `index.html` が表示されます。

## app-ads.txt

AdMob 用の `app-ads.txt` はリポジトリ直下に置いてあります。
公開後、以下のようにアクセスできればOKです。

```text
https://あなたのドメイン/app-ads.txt
```

## ページ構成

- ホーム：ODORU STUDIO ロゴとアプリ一覧
- ThinkBlank：アイコン、App Storeリンク、説明文、サポート情報
